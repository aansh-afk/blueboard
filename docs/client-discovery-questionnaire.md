## 2) What Are You Building?

### Project Type (check one or more)

- [ ] Marketing website (brand, credibility, lead generation)
- [ ] SaaS product (web app)

- [ ] E-commerce

### Core Goal (pick top 1-2)

- [ ] Generate leads

- [ ] Educate users
- [ ] Launch MVP quickly
- [ ] Improve existing UX/UI
- [ ] Rebrand / reposition

### Success Metrics

- How will we know this project is successful? (examples: signups/month, demo requests, trial-to-paid conversion, revenue, lower churn)

## 3) Scope and Deliverables

### If Marketing Website

- Which pages are needed? (Home, Pricing, About, Contact, Blog, FAQ, etc.)
- Do you need landing pages for ads/campaigns?
- Do you need CMS/blog editing capability?
- Do you need multilingual support?

### If SaaS Product

- What is the core workflow users must complete?
- User roles needed? (admin, manager, member, guest)
- Auth required? (email/password, Google, SSO)
- Billing needed? (subscriptions, usage-based, invoices)
- Key features needed at launch? (dashboard, analytics, forms, automation, etc.)
- Integrations needed? (Stripe, Slack, HubSpot, Notion, etc.)

### MVP vs Future

- What must be in **V1 launch**?
- What can wait for **phase 2**?
